# Legacy Russian Course Content

This directory contains the original Russian-language course materials that served as the foundation for the current English course.

## Status

**Legacy/Historical** - This content is preserved for historical reference and to honor the project's origins. It is not actively maintained.

## About

This course was originally created by Vladyslav Ponomaryov and contributors. The current project (pymastery-vp) is a derivative work that translates and adapts this content for English-speaking learners.

**Upstream Repository:** https://github.com/PonomaryovVladyslav/PythonCourses.git

## Structure

- `lessons/` - Original lesson files (`.md` format)
- `pictures/` - Images and diagrams for legacy lessons

## Active Content

For the current, actively maintained English course content, see `content/en/`.
